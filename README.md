## 开始使用
1. 导入项目根目录下的 sql 数据，`xpress-data.sql`包含测试数据，`xpress.sql`只创建表。
2. 配置`application.yaml`数据源。
3. 运行 Spring Boot
4. 访问后台管理面板`http://yourdomain/admin/index.html`

## 技术栈
- Spring Boot
- MyBatis-Plus
- Vue.js
